# 현재 시간 입력

|         | excel            | google sheet        |   |   |
|---------|------------------|---------------------|---|---|
| windows | ctrl + shift + ; | ctrl + shift + ;    |   |   |
| mac     | command + ;      | command + shift + ; |   |   |
